HTML Text Formatting Elements

This page shows common HTML text formatting tags and examples.

Bold Text

The b tag makes text bold.

Example: This text is bold

Strong Text

The strong tag shows important bold text.

Example: This text is important

Italic Text

The i tag makes text italic.

Example: This text is italic

Emphasized Text

The em tag emphasizes text.

Example: This text is emphasized

Underline Text

The u tag underlines text.

Example: This text is underlined

Marked Text

The mark tag highlights text.

Example: This text is highlighted

Small Text

The small tag makes text smaller.

Example: This text is small

Deleted Text

The del tag shows deleted text.

Example: This text is deleted

Inserted Text

The ins tag shows inserted text.

Example: This text is inserted

Subscript Text

The sub tag shows subscript text.

Example: H2O
